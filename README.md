Project Title
# Windows 10 File Management and Folder Permissions in Oracle VirtualBox

## Overview
This project demonstrates **basic File Management tasks** and **folder permission setup** in Windows 10 using Oracle VirtualBox. It includes step-by-step instructions, screenshots, and notes.

## Tasks Covered
1. Create a new folder
2. Copy files
3. Move files
4. Delete files
5. Set folder permissions (Read, Write, Full Control)

## Step-by-Step Instructions

### 1. Create a New Folder
- Right-click on Desktop or in File Explorer → New → Folder  
- Enter the folder name  

### 2. Copy Files
- Right-click the file → Copy → Paste in the desired location  
- Or use **Ctrl + C** and **Ctrl + V**

### 3. Move Files
- Right-click the file → Cut → Paste in the desired location  
- Or use **Ctrl + X** and **Ctrl + V**

### 4. Delete Files
- Right-click the file → Delete → Confirm  
- Or select file → Press **Delete** key → Confirm

### 5. Set Folder Permissions
1. Right-click the folder → Properties → Security tab  
2. Select a user or group (or add a new one if needed)  
3. Check the permissions you want to assign:  
   - **Read** → View files/folders only  
   - **Write** → Modify files/folders  
   - **Full Control** → Read, write, modify, and delete files/folders  
4. Click **Apply → OK** to save changes
