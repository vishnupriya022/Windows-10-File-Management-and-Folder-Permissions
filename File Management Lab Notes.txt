File Management Tasks in Windows 10 VM:

1. Create a New Folder
- Right-click Desktop/File Explorer → New → Folder → Enter name

2. Copy Files
- Right-click file → Copy → Paste
- Or use Ctrl + C and Ctrl + V

3. Move Files
- Right-click file → Cut → Paste
- Or use Ctrl + X and Ctrl + V

4. Delete Files
- Right-click file → Delete → Confirm
- Or select file → Press Delete key → Confirm

Folder Permission Tasks in Windows 10 VM:

1. Open Folder Properties
- Right-click the folder → Properties → Security tab

2. Select User or Group
- Under “Group or user names,” select a user
- To add a new user → Click Edit → Add → Enter username → OK

3. Set Permissions
- Check the boxes for the desired permission:
  - Read → View files/folders only
  - Write → Modify files/folders
  - Full Control → Read, write, modify, and delete files/folders

4. Apply Changes
- Click Apply → OK → Permissions are saved
